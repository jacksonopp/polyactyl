---
title: Keyboard Shortcuts
description: Complete reference of all keyboard shortcuts in Polyactyl.
---

Polyactyl is designed to be keyboard-first. All common actions have shortcuts so you can stay in flow without reaching for the mouse.

## Global

| Action | macOS | Windows / Linux |
|---|---|---|
| Open folder | <kbd>⌘</kbd><kbd>O</kbd> | <kbd>Ctrl</kbd><kbd>O</kbd> |
| Command palette (file search) | <kbd>⌘</kbd><kbd>P</kbd> | <kbd>Ctrl</kbd><kbd>P</kbd> |
| Command palette (command mode) | <kbd>⌘</kbd><kbd>Shift</kbd><kbd>P</kbd> | <kbd>Ctrl</kbd><kbd>Shift</kbd><kbd>P</kbd> |
| Command palette (file browser) | <kbd>⌘</kbd><kbd>K</kbd> | <kbd>Ctrl</kbd><kbd>K</kbd> |
| Save current file | <kbd>⌘</kbd><kbd>S</kbd> | <kbd>Ctrl</kbd><kbd>S</kbd> |
| Close active tab | <kbd>⌘</kbd><kbd>W</kbd> | <kbd>Ctrl</kbd><kbd>W</kbd> |
| Switch to next tab | <kbd>⌘</kbd><kbd>]</kbd> | <kbd>Ctrl</kbd><kbd>]</kbd> |
| Switch to previous tab | <kbd>⌘</kbd><kbd>[</kbd> | <kbd>Ctrl</kbd><kbd>[</kbd> |

### Command Palette Modes

- **<kbd>⌘</kbd><kbd>P</kbd> / <kbd>Ctrl</kbd><kbd>P</kbd>** — Opens the command palette in **file search mode**. Start typing any part of a filename to fuzzy-search and switch to it instantly.
- **<kbd>⌘</kbd><kbd>Shift</kbd><kbd>P</kbd> / <kbd>Ctrl</kbd><kbd>Shift</kbd><kbd>P</kbd>** — Opens the command palette in **command mode**. Run app-level actions without touching the mouse.
